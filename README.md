Project Summary

A modern, responsive web platform for E-Cell, University of Lucknow, designed to connect students with alumni mentors and showcase events, impact, and entrepreneurial culture.

Key Design Decisions

Color Palette:
Navy blue for trust and professionalism, bright blue for actions, and purple for creativity. Dark mode by default with light mode support for accessibility.

Typography:
Poppins for headings (bold & modern) and Inter for body text (clean & readable).

Layout:
Responsive, card-based design using Tailwind CSS. Mobile-first approach with clean spacing and rounded components.

Visual Style:
Subtle grid backgrounds, gradient blurs, and smooth animations to create a modern, premium feel without distraction.

UX Highlights

Sticky navigation with theme toggle

Interactive alumni cards with expertise tags and LinkedIn links

Event gallery with auto-scrolling visuals and stats

Simple modal forms with validation and toast feedback

Light/Dark mode with saved user preference
----------------------------------------------------------------------------------


*Problem:** 
Students struggle to find and connect with relevant alumni mentors who have expertise in their areas of interest.

**Solution:**
- Curated alumni directory with clear expertise tags
- Direct LinkedIn links for verification
- One-click mentorship request system
- Filtering by expertise (FinTech, EdTech, AgriTech, etc.)
- Success stories and quotes build credibility

*Problem:**
Past event highlights are often buried in social media feeds or forgotten, making it hard to showcase E-Cell's impact.

**Solution:**
- Dedicated gallery page with visual storytelling
- Auto-scrolling photo carousels for passive engagement
- Detailed event descriptions with dates and impact
- Statistics dashboard showing overall reach
- Downloadable brochure for comprehensive information
